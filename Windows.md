# Building on Windows

## Setup CYGWIN

1. Download the 64-bit version from https://www.cygwin.com/
2. Install the following packages:
    - libgtk3-devel
    - libboost-devel
    - libboost_filesystem1.66
    - libboost_system1.66
    - libsqlite-devel
    - make
    - g++
3. Run `make` or `make build-run` in the directory where the project is cloned.
